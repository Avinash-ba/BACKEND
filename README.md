Task Comments API (Flask Backend)
Overview
This project implements RESTful APIs for managing comments associated with tasks.
It is built using Flask and SQLite and follows proper CRUD principles.
Automated tests are included using pytest.

Tech Stack
-Python 3
-Flask
-Flask-SQLAlchemy
-SQLite
-Pytest

Features
-Create comments for a task
-Retrieve comments for a task
-Update a comment
-Delete a comment
-Automated test coverage for all APIs
